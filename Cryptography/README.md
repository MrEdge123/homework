# 密码学作业

## 文件目录

- `cryptProgram`: 项目文件夹, 由QT构建
- `cryptProgram.exe`: 主程序, 已打包, 可直接执行

## 已完成模块

- 异或密码
- DES算法：S盒输入输出情况
- DES算法：改变位的情况
- DH密钥交换协议
- ELGamal
- RSA算法
- 短块加密
